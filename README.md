# QuickStore
Esse é um projeto de aplicação de sistemas distribuidos para uma aplicação web de e-commerce de gerenciamento de pedidos.

## Especificações
##
#### O sistema conta  com 6 módulos
* Cliente
* Usuário
* Pedido
* Entrega
* Produto
* Banco de Dados
